Wi-Fi Security

The Process of Wi-Fi

Let’s talk about the thing we constantly connect to. The Wi-Fi! We are going to talk about Wi-Fi Security. Before discussing Wi-Fi Security, let’s look at what Wi-Fi is. Wi-Fi stands for Wireless Fidelity. Wireless Fidelity allows the creation of a wireless local area network (WLAN). (STL) Local Area Network (LAN) is “a computer networking solution that connects devices in a limited area, such as a building, office, home and even campuses of adjacent buildings.” (TPX) Wi-Fi is a wireless networking technology that connects computers, tablets, phones, and other devices to the Internet. (Verizon and Cisco 1) The Wi-Fi’s process must follow the IEEE 802.11 standard. This “standard defines the protocols that enable communications with current Wi-Fi-enabled wireless devices, including wireless routers and wireless access points.” (Cisco 1) Wireless access points allow wireless devices to connect to wireless networks. The user can connect to a wireless router. A wireless router is an Internet service provider product used to connect to a cable or an Internet network. (Cisco 1) Now we understand each component, we can discuss Wi-Fi Security.

Wi-Fi Security is the “protection of devices and networks connected in a wireless environment.” (Cisco 2) With Wi-Fi security, there are four types of wireless security protocols. The four types are WEP, WPA, WPA2, and WPA3. Let’s break down each type. WEP, also known as Wired Equivalent Privacy, was the first wireless security protocol to provide standard methods for network security from 1997 to 2004. Then WPA, also known as Wi-Fi Protected Access, dealt with the flaws WEP presented. WPA 2 was developed in 2004, handling network security better than WPA. WPA 3 brings different changes to wireless security. (Cisco 2 and SecureW2) With these four types in mind, how can we practice Wi-Fi security? A common practice is practicing password security. Other practices you can take are verifying the correct wifi, turning off automatic connection, not shopping on public networks, using VPN, and more found on San Diego State University’s page.
